A real-time monitoring dashboard for locomotive parameters such as temperature, fuel level, and vibration.

## Features
- Live data simulation
- Real-time dashboard (Chart.js)
- Flask backend APIs
- SQLite database
- Threshold-based monitoring concept

##  Tech Stack
- Python (Flask)
- SQLite
- JavaScript (Fetch API)
- Chart.js
